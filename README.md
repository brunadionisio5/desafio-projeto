## ⚠️ Observação sobre a execução do projeto

Durante o desenvolvimento deste desafio, foi realizada a criação do banco de dados relacional no MySQL, incluindo a estrutura de tabelas, relacionamentos e inserção de dados conforme o modelo proposto.

A integração com o Azure MySQL não pôde ser concluída devido a dificuldades técnicas na configuração do serviço (incluindo conexão e regras de firewall).

Como alternativa, o banco de dados foi executado localmente utilizando MySQL Workbench, permitindo a realização das etapas de modelagem, inserção de dados e preparação para análise.

A etapa de integração com o Power BI foi iniciada, porém não foi possível concluir a conexão com a instância MySQL no ambiente utilizado.

Mesmo assim, todo o processo de modelagem relacional, criação de tabelas, relacionamentos e inserção de dados foi implementado e validado localmente.
